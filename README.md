# Modélisation 3D - TP1
## Installation
- Créer un nouveau projet Unity (version >= à 6000.2.3f1)
- Télécharger l'unitypackage et l'importer dans Unity => menu Assets > Import Package > Custom Package
- Ouvrir la scène "Mode_3d" dans le dossier Assets > Scenes

Chaque exercice possède son propre GameObject et son propre Script
